# EFFECTS_QUICK_REFERENCE

Compact lookup: effect name + constructor signature, organized by category.
For detailed descriptions, targeting info, and examples, see EFFECTS_INDEX.md.

**How to use:** Search this file for keywords from the card text (e.g. "graveyard", "life", "shuffle", "destroy"). Once you find a candidate effect, grep EFFECTS_INDEX.md for its name to get full details.

- `RegisterDelayedReturnSourceTransformedEffect()` ŌĆö ON_DEATH effect that registers a delayed end-step return from the source card's owner's graveyard to the battlefield transformed. Used by Loyal Cathar-style "When this dies, return it transformed at the beginning of the next end step." `(true, true)` = only on sacrifice + under owner's control (Biolume Egg).
- `RegisterDelayedCreateTokenEffect(CreateTokenEffect tokenEffect)` ŌĆö registers a delayed trigger that resolves `tokenEffect` at the beginning of the next end step, creating the token(s) under the resolving controller's control. Used by Rukh Egg-style "When this dies, create a ... token at the beginning of the next end step."
- `NecropotenceSetAsideTopCardEffect()` ŌĆö `CardDrawingEffect` (1). Activated-ability effect (Necropotence `Pay 1 life:`): exile the top library card face down, then return it to hand at the **controller's own** next end step via a `ReturnExiledCardToHandAtEndStep` delayed action (drained in `StepTriggerService.handleEndStepTriggers`, `activePlayerId == ownerId`). Pair with `PayLifeCost(1)` in the ability's effect list.
- `ExileDiscardedCardFromGraveyardEffect()` ŌĆö `ON_CONTROLLER_DISCARDS` trigger marker: "whenever you discard a card, exile that card from your graveyard" (Necropotence). Collected inline in `DiscardTriggerCollectorService` (moves the just-discarded card from graveyard to exile); no-op if it isn't in the graveyard.
- `RegisterDelayedReturnDamagedCreatureUnderControlEffect()` ŌĆö `ON_DAMAGED_CREATURE_DIES` effect (Seraph). Returns the creature that died onto the battlefield under your control at the next end step (via `DelayedGraveyardToBattlefieldUnderControl`, drained in `StepTriggerService`), with a PERMANENT control effect so a stolen opponent's creature stays yours yet dies to its owner's graveyard; linked creatures are sacrificed when their controller loses control of the still-present Seraph (`StateBasedActionService`). Reads the dying card id from `StackEntry.triggeringCardId`.
- `RegisterDrawCardsAtNextUpkeepEffect(int count)` / `()` ŌĆö `CardDrawingEffect`; schedules the resolving controller to draw `count` cards (default 1) at the beginning of the next turn's upkeep via a `DrawCardsAtNextUpkeep` delayed action. Used by Blessed Wine "Draw a card at the beginning of the next turn's upkeep."
- `RegisterTransformSourceAtNextUpkeepEffect()` ŌĆö ON_ALLY_CREATURE_DIES registrar (Archangel Avacyn). Queues a `TransformSourceAtNextUpkeep` delayed action; at the next upkeep (any player) pushes `TransformToBackFaceEffect` onto the stack (no-op if already transformed / gone).
- `TransformToBackFaceEffect()` ŌĆö transform source to back face only if still front-face; no-op if already transformed
- `MeldWithNamedCreatureEffect(String partnerName)` ŌĆö exile source + owned/controlled creature named `partnerName`, meld into source `backFaceCard` (Gisela ŌåÆ Brisela). Track components on `Permanent.meldComponentCards`.
- `RegisterDamageAtNextUpkeepUnlessPaysEffect(int damage, String manaCost)` ŌĆö SPELL registrar (Quenchable Fire). Piggybacks on the accompanying `DealDamageToTargetPlayerOrPlaneswalkerEffect`'s target (declares no target of its own) and schedules a `DamageAtNextUpkeepUnlessPays` delayed action, drained in `StepTriggerService.handleUpkeepTriggers` at the **spell controller's** next upkeep. The paying party (targeted player, or targeted planeswalker's controller) may pay `manaCost` or the target takes `damage` (pushed as a `DamageTargetPlayerOrPlaneswalkerUnlessPaysEffect` may-pay trigger). Fizzles if the target is gone. Pair with `DealDamageToTargetPlayerOrPlaneswalkerEffect(damage)` in the same SPELL slot for the immediate damage.
- `RegisterCombatDamageReflectionEffect()` ŌĆö "This turn, whenever an attacking creature deals combat damage to you, it deals that much damage to its controller" (Harsh Justice). Registers a `DelayedCombatDamageReflection` for the rest of the turn; `CombatDamageService` reflects each attacking creature's player-damage back to its controller. Pair with `setSpellCastTimingRestriction(DECLARE_ATTACKERS_IF_ATTACKED)`.

## Targeting rules (summary)

- An effect declares targeting by overriding `targetSpec()` to a non-NONE `TargetSpec` (category + `harmful` flag + optional `PermanentPredicate`); the deleted legacy `canTarget*` booleans derived from it. See `EFFECTS_INDEX.md` ┬¦ "Effect targeting declarations".
- Effects in SPELL slot whose `targetSpec().category()` includes players force player targeting at cast time; those including permanents force permanent targeting (and the category/predicate type-check it).
- ETB/triggered/saga slots: targeting declarations don't force spell-level targeting.
- `CostEffect` subtypes are excluded from targeting computation.
- Targeting is computed by `EffectResolution.needsTarget(card)` / `needsSpellCastTarget(card)`.
- Never call `setNeedsTarget`/`setNeedsSpellTarget` directly.

## Capability / marker interfaces

DESCRIPTIVE interfaces in `model/effect/` that a family of effects implements so the AI
(`SpellEvaluator`, `InstantCategoryClassifier`, `AiTargetSelector`, `HardAiDecisionEngine`,
`GameSimulator`, `AiManaManager`) can read a FACT about the effect instead of `instanceof`-ing each
concrete type. **When you add a new effect that fits one of these families, implement the interface**
(return existing record components ŌĆö the impl is purely additive) so the AI scores/targets it without
a code change. Interfaces are auto-exempt from `EffectDispatchRatchetTest`.

- `CostEffect` ŌĆö additional costs (sacrifice, discard, exile, counter removal, tap creature). AI
  cost-valuation facets describe the resource paying gives up (default to neutral; override only the
  one that matches the record): `consumedPermanentFilter()` (`PermanentPredicate` selecting a
  payer-chosen battlefield permanent to sacrifice/return ŌĆö creature/artifact/filtered),
  `consumesSourcePermanent()` (sacrifice this permanent), `sacrificesChosenCreature()` (plain
  "sacrifice a creature"), `lifePaid(currentLife)`, `sourceCountersRemoved()`,
  `consumedGraveyardCardCount()` + `consumedGraveyardCardType()`. Read by `GameSimulator`
  (payment-planning: find a sacrifice target / graveyard cards to exile) and
  `HardAiDecisionEngine.evaluateAbilityCosts` (score the cost). Overridden by `SacrificeCreatureCost`,
  `SacrificeArtifactCost`, `SacrificePermanentCost`, `SacrificeSelfCost`, `PayLifeCost`,
  `RemoveChargeCountersFromSourceCost`, `ExileNCardsFromGraveyardCost`; all other cost records inherit
  the neutral defaults (the AI never reasoned about them). Payment EXECUTION stays concrete in
  `AbilityActivationService`.
- `ManaProducingEffect` ŌĆö mana abilities (CR 605.1a). AI-estimator facets (default to neutral;
  override only if the AI should model the mana): `estimatedManaColor()`, `estimatedManaAmount()`,
  `estimatedCountsAllColors()`, `estimatedWildcardMana()`, `modeledByManaEstimator()`
- `DamageDealingEffect` ŌĆö deals a `DynamicAmount` to one target category; `damageAmount()`,
  `canDamageCreatures()`, `canDamagePlayers()`. Impl `DealDamageToAnyTargetEffect`,
  `DealDamageToTargetCreatureEffect`, `DealDamageToPlayersEffect`
- `GlobalDamageMultiplyingEffect` ŌĆö static permanent that scales ALL damage in the game regardless of
  source/target/controller; `damageMultiplierFactor()` (2 for a doubler, stacks multiplicatively). Read
  by `GameQueryService.getDamageMultiplier` and the Myr Battlesphere damage path. Impl `DoubleDamageEffect`
  (Furnace of Rath). NOT for the selective doublers (`DoubleControllerDamageEffect`,
  `DoubleDamageToEnchantedPlayerEffect`, `DoubleEquippedCreatureCombatDamageEffect`) ŌĆö each selects a
  different permanent subset and stays concrete in its own dedicated query method
- `RemovalEffect` ŌĆö single-target destroy/exile/bounce; `removalKind()` returns `RemovalKind`
  (`DESTROY`/`EXILE`/`BOUNCE`) or `null` when not single-target removal (e.g. mass bounce).
  Impl `DestroyTargetPermanentEffect`, `ExileTargetPermanentEffect`,
  `ExileTargetCreatureAndSameNameFromHandIfGodEffect` (Hour of Glory),
  `ReturnTargetPermanentToHandWithManaValueConditionalEffect`, `ReturnToHandEffect` (TARGET scope)
- `BoardWipeEffect` ŌĆö recognition-only fact for board sweeps; `sweepsBoard()` (true when the config is
  a mass sweep). Lets the AI's phase/pressure multipliers detect a wipe without naming the type; the
  per-type wipe SCORING stays concrete (each sweep is valued differently). Impl `MassDamageEffect`
  (always), `DestroyAllPermanentsEffect` (always), `ReturnToHandEffect` (only `BounceScope.ALL_MATCHING`)
- `CardDrawingEffect` ŌĆö `drawnCardAmount()` (`DynamicAmount`). Impl `DrawCardEffect`
- `LifeGainEffect` ŌĆö `lifeGainAmount()` (`DynamicAmount`). Impl `GainLifeEffect`, `GainLifeEqualToTargetCreaturePowerEffect`
- `TokenCreatingEffect` ŌĆö `tokenAmount()`, `tokenType()`, `tokenPower()`, `tokenToughness()`.
  Impl `CreateTokenEffect`
- `CreatureBoostEffect` ŌĆö targeted P/T pump; `powerBoost()`, `toughnessBoost()` (`DynamicAmount`).
  Impl `BoostTargetCreatureEffect`
- `StaticCreatureBoostEffect` ŌĆö continuous anthem/aura boost; `powerBoost()`, `toughnessBoost()`
  (int), `grantedKeywords()`, `scope()`, `filter()`. Impl `StaticBoostEffect`. The
  `StaticBoostEffect(int, int, GrantScope, CounterType scalingCounter)` ctor multiplies the flat
  boost by the count of that counter on the source ("All creatures get +1/+0 for each time counter
  on this" ŌĆö Infinite Hourglass, `GrantScope.ALL_CREATURES`, `CounterType.TIME`)
- `KeywordGrantingEffect` ŌĆö `keywords()`, `scope()`, `filter()` (predicate narrowing the scope, or
  `null`). Impl `GrantKeywordEffect`
- `ControlStealingEffect` ŌĆö `controlDuration()`. Impl `GainControlOfTargetEffect`, `StealDyingOpponentPermanentUnlessPaysLifeEffect`
- `CounterSpellingEffect` ŌĆö marker for "counter target spell". Impl `CounterSpellEffect`,
  `CounterUnlessPaysEffect`, `CounterEachTargetSpellEffect`
- `CounterUnlessEffect` ŌĆö "counter unless controller pays a ransom" family; `ransomKind()`
  (`PAY_MANA`/`DISCARD_CARD`) + `ransomMagnitude()` (mana amount, or 1 for a discard). Lets the
  trigger-collection and may-ability choice-flow services recognise the family and route on kind
  without `instanceof`-ing each variant (the pay/discard orchestration itself stays in the engine).
  Impl `CounterUnlessPaysEffect`, `CounterUnlessDiscardsEffect`
- `ClashOutcomeConditionalEffect` ŌĆö clash-trigger effect applied only on one outcome; `wrapped()` +
  `appliesOnWin()`. Read by `TriggerCollectionService.fireClashTriggers` to resolve the won/lost
  clause without branching on the two wrapper types. Impl `IfWonClashEffect`, `IfLostClashEffect`
- `RegenerationEffect` ŌĆö marker for regeneration. Impl `RegenerateEffect`
- `ProtectionGrantingEffect` ŌĆö static "protection from ŌĆ”" facts read by the engine query layer
  (`GameQueryService`), not the AI: `protectionFromColors()` + `protectionScope()`
  (`GrantScope` null = self / `EQUIPPED_CREATURE`), `protectionFromCardTypes()`,
  `protectionFromSubtypes()`, `protectionFromManaValueAtLeast()` (`OptionalInt`),
  `protectsFromEverything()` (Progenitus ŌĆö protection from every source; checked in
  `hasProtectionFromSourceCardTypes` since every source has a card type). A record fills only
  its facet and inherits the empty/self defaults. Impl `ProtectionFromColorsEffect`,
  `ProtectionFromCardTypesEffect`, `ProtectionFromSubtypesEffect`, `ProtectionFromManaValueEffect`,
  `ProtectionFromEverythingEffect`.
  NOT for runtime-only protection: chosen-color (`ProtectionFromChosenColorEffect`, a
  `ChooseColorEffect`) or "protection from non-[subtype] creatures" (tracked on the `Permanent`)
- `BlockabilityRestrictionEffect` ŌĆö attacker-side evasion facts read by the engine query layer
  (`GameQueryService`): `cantBeBlocked()`, `unblockableIfDefenderControls()` (`PermanentPredicate`),
  `unblockableIfControllerCastHistoricSpellThisTurn()`, `unblockableWhileAttackingAlone()`,
  `blockableOnlyBy()` + `blockableOnlyByDescription()`, `cantBeBlockedByCreaturesMatching()`. The engine
  evaluates the predicates and supplies engine-computed facts (attacking-alone, historic-spell). Impl
  `CantBeBlockedEffect`, `CanBeBlockedOnlyByFilterEffect`, `CantBeBlockedByCreaturesMatchingPredicateEffect`,
  `CantBeBlockedIfDefenderControlsMatchingPermanentEffect`,
  `CantBeBlockedIfControllerCastHistoricSpellThisTurnEffect`, `CantBeBlockedIfAttackingAloneEffect`
- `BlockingRestrictionEffect` ŌĆö blocker-side restriction facts (`GameQueryService`): `cantBlock()`,
  `canBlockOnlyAttackersMatching()` + `canBlockOnlyAttackersDescription()`, and the board-wide
  `globalCantBlockBlockerMatcher()` / `globalCantBlockAttackerMatcher()` / `globalCantBlockDescription()`.
  Impl `CantBlockEffect`, `CanBlockOnlyIfAttackerMatchesPredicateEffect`,
  `MatchingCreaturesCantBlockMatchingCreaturesEffect`
- `BlockCostEffect` ŌĆö per-block mana tax fact (`CombatBlockService.declareBlockers`):
  `blockCost(int attackerPower)` = generic mana to declare the block (0 = free). Impl
  `CantBlockHighPowerCreaturesUnlessPaysEffect`
- `AttackOrBlockRestrictionEffect` ŌĆö combined "can't attack or block" facts (attack side in
  `CombatAttackService`, block side in `GameQueryService`): `globallyCantAttackOrBlock()`
  (`PermanentPredicate`), `cantAttackOrBlockUnless()` (`Condition`), `restrictionDescription()`. Impl
  `MatchingCreaturesCantAttackOrBlockEffect`, `CantAttackOrBlockUnlessEffect`
- `PermanentLockEffect` ŌĆö per-permanent, duration-scoped lock facts read off a `FloatingContinuousEffect`
  whose `affectedPermanentId` is the locked permanent: `locksAttacking()` (`CombatAttackService`),
  `locksBlocking()` (`GameQueryService.findBlockDenial`), `locksActivatedAbilities()`
  (`AbilityActivationService.validateNotBlockedByStaticAbilityLock`; mana abilities included, triggered
  abilities unaffected). Queried via `GameQueryService.isLockedFrom{Attacking,Blocking,ActivatingAbilities}`.
  Impl `LockTargetPermanentEffect(locksAttacking, locksBlocking, locksActivatedAbilities, duration)` ŌĆö
  the resolved effect that stamps the floating lock on its target creature (Detain / "can't attack this
  turn", Edifice of Authority). `UNTIL_END_OF_TURN` expires at cleanup, `UNTIL_YOUR_NEXT_TURN` at the
  ability controller's next turn start.
- `OtherCreaturesCantAttackThisTurnEffect()` ŌĆö SPELL one-shot (Intimidation Bolt): every creature
  except the accompanying damage spell's target can't attack this turn. Appends the target
  (`entry.getTargetId()`) to the turn-scoped `GameData.otherCreaturesCantAttackExemptCreatureIds`, read
  in `CombatAttackService.canCreatureAttack` (so it also bars creatures entering later; a dead target
  ŌćÆ nobody attacks). Cleared at each turn transition. Pair with `DealDamageToTargetCreatureEffect(N)`
- `NoDefenderAttackPermissionEffect` ŌĆö "attack as though no defender" facts
  (`GameQueryService.canAttackDespiteDefender`): `grantsCarrierAttackAsThoughNoDefender()` (self grant,
  incl. `ConditionalEffect`-wrapped and floating grants) and `noDefenderAttackMatcher()`
  (`PermanentPredicate`, board-wide). Impl `CanAttackAsThoughNoDefenderEffect`,
  `MatchingCreaturesCanAttackAsThoughNoDefenderEffect`
- `CombatDamageTriggerContextEffect` ŌĆö how `CombatDamageService` should populate a fired
  `ON_COMBAT_DAMAGE_TO_PLAYER` trigger's stack entry: `combatDamageTriggerContext()` returns a
  `TriggerContext` ŌĆö `DAMAGED_PLAYER_WITH_DAMAGE_AMOUNT` (xValue = damage dealt, targetId = damaged
  player, no source; Balefire Dragon), `SOURCE_SELF` (source = the dealing creature, no target;
  explore / self-counter), `DAMAGED_PLAYER` (targetId = damaged player, source = the dealing creature;
  "that player discards/mills/ŌĆ”") ŌĆö or `null` for the plain no-context entry (also the default for
  effects that don't implement it). Recipient-dependent effects return `DAMAGED_PLAYER` only when their
  recipient is the target player and `null` otherwise. Impl `ReturnPermanentsOnCombatDamageToPlayerEffect`,
  `DealDamageToEachCreatureDamagedPlayerControlsEffect`,
  `LookAtTopXCardsPermanentsToBattlefieldRestToGraveyardEffect`, `PutCountersOnSourceEffect`,
  `RemoveAllCountersFromSelfEffect`, `ExploreEffect`, `ExileTopCardsRepeatOnDuplicateEffect`,
  `TargetPlayerRandomDiscardOrControllerDrawsEffect`, `RevealRandomCardFromTargetPlayerHandEffect`,
  `RevealRandomCardFromTargetPlayerHandLoseLifeEqualToManaValueEffect` (reveal random + lose life = its mana value, Singe-Mind Ogre),
  `LookAtRandomCardInTargetPlayerHandEffect`,
  `SphinxAmbassadorEffect`, `TargetPlayerExilesFromHandEffect`, `ChooseCardsFromTargetHandEffect`,
  `SkipNextCombatPhaseEffect`, `TargetPlayerCantGainLifeRestOfGameEffect`, `DiscardEffect` /
  `MillEffect` / `DealDamageToPlayersEffect` (only when recipient is the target player). NOT for the
  bespoke `MayEffect`-wrapped or destroy/sacrifice "damaged player controls" flows, which stay in
  `CombatDamageService`
- `GraveyardAbilityGrantingEffect` ŌĆö static "each creature card in your graveyard has '[graveyard
  activated ability]'" facts read by `GameQueryService.computeGrantedGraveyardAbilitiesForOwnedCreatureCard`
  (and appended to the effective graveyard abilities by `AbilityActivationService` + the graveyard
  `CardView`): `grantedGraveyardAbility()`. Impl `GrantGraveyardAbilityToCreatureCardsEffect` (Sedris,
  the Traitor King ŌĆö unearth {2}{B})
- `CombatOpponentReferencingEffect` ŌĆö marker for an effect that, when fired from an **Aura/Equipment**
  combat trigger (`ON_BLOCK` / `ON_BECOMES_BLOCKED PER_BLOCKER`), acts on the enchanted/equipped
  creature's combat opponent. `CombatTriggerService` bakes that opponent in as the trigger's
  non-targeting `targetId` (so it can't fizzle) instead of prompting the controller for a target. Impl
  `DestroyCombatOpponentAtEndOfCombatEffect` (Venom ŌĆö Basilisk-style Aura, non-Wall filter),
  `SkipNextUntapEffect(TapUntapScope.TARGET)` (Shield of the Righteous ŌĆö equipped creature blocks ŌåÆ
  blocked attacker skips next untap)
- `FreeCyclingEffect` ŌĆö static "you may pay {0} rather than pay cycling costs" fact read by
  `AbilityActivationService` at cycling activation (CR 118.9 alt cost; only the mana cost is replaced,
  the discard is still paid): `minCardsInHand()` cards the controller must hold, counting the card

## Wrapper / modifier effects

Core wrappers (all take `CardEffect wrapped` as first/only effect arg):
- `SequenceEffect.of(step1, step2, ŌĆ”)` (>=2 steps) ŌĆö resolve the steps strictly in order, as one effect (engine splices them into the entry's live effect list; NO "if you do" contingency, a no-op step doesn't stop later steps). Use ONLY to gate several steps under a single-effect wrapper (`MayEffect(SequenceEffect.of(a, b))`, `ConditionalEffect(cond, SequenceEffect.of(a, b))`, `FlipCoinWinEffect(SequenceEffect.of(a, b))`) or to keep several steps one atomic triggered ability on a trigger slot. Do NOT wrap a plain spell/ability effect list ŌĆö use flat `addEffect(...)` calls instead (Act of Treason / Drain Life pattern). Single target is shared by every targeting step; multi-target groups inside a sequence are unsupported
- `MayEffect(CardEffect, String prompt)` ŌĆö "you may"
- `MayPayManaEffect(String manaCost, CardEffect, String prompt)` ŌĆö "you may pay {X}"
- `MayPayTapPermanentsEffect(TapMultiplePermanentsCost, CardEffect, String prompt)` ŌĆö "you may tap N permanents"
- `ConditionalEffect(new Metalcraft(), CardEffect)` ŌĆö 3+ artifacts
- `ConditionalEffect(new SpellManaSpentAtLeast(minMana), wrapped)` ŌĆö mana spent to cast triggering spell >= N
- `ConditionalEffect(new SpellXAtLeast(minX), wrapped)` ŌĆö resolving spell/ability's chosen X (entry `xValue`) >= N (Martial Coup "If X is 5 or more")
- `ConditionalEffect(new Morbid(), CardEffect)` ŌĆö creature died this turn
- `ConditionalEffect(new Raid(), CardEffect)` ŌĆö attacked this turn
- `ConditionalEffect(new ControllerCastAnotherSpellThisTurn(filter), wrapped)` ŌĆö another spell matching filter cast this turn (excludes resolving spell)
- `ConditionalEffect(new NotCondition(inner), wrapped)` ŌĆö logical negation of any condition ("unless ŌĆ”"), e.g. Hotheaded Giant enters with -1/-1 counters `new NotCondition(new ControllerCastAnotherSpellThisTurn(new CardColorPredicate(RED)))`
- `TriggeringCardConditionalEffect(CardPredicate, CardEffect)` ŌĆö triggering card matches predicate
- `TriggeringPermanentConditionalEffect(PermanentPredicate, CardEffect)` ŌĆö triggering permanent matches predicate
- `ConditionalEffect(new ControlsAnotherPermanent(filter), wrapped)` ŌĆö controls another matching permanent
- `ConditionalEffect(new ControllerLifeAtLeast(threshold), wrapped)` ŌĆö life >= N
- `ConditionalEffect(new ControllerHasMoreLifeThanAnOpponent(), wrapped)` ŌĆö you have strictly more life than at least one opponent (Feudkiller's Verdict)
- `ConditionalEffect(new TargetPermanentMatches(PermanentPredicate), wrapped)` ŌĆö the targeted permanent matches the predicate at resolution (e.g. "destroy target permanent if it's red": wrap `DestroyTargetPermanentEffect` with `PermanentColorInPredicate(Set.of(RED))`). Targeting stays unrestricted (any permanent); the color check is a resolution-time intervening-if. Hydroblast mode 1
- `ConditionalEffect(new TargetSpellMatches(StackEntryPredicate), wrapped)` ŌĆö the targeted spell on the stack matches the predicate at resolution (e.g. "counter target spell if it's red": wrap `CounterSpellEffect` with `StackEntryColorInPredicate(Set.of(RED))`). Targeting stays unrestricted (any spell); the color check is a resolution-time intervening-if. Hydroblast mode 0

Metalcraft / Morbid / Raid / ControlsAnotherPermanent / OpponentControlsMoreLands are ETB trigger gates
(`Condition.isEtbTriggerGate()`): a targeted ETB wrapped in one never asks for its target at cast
time ŌĆö the target is chosen as the trigger goes on the stack (see TRIGGER_SLOT_TARGETING.md,
"ON_ENTER_BATTLEFIELD targeted triggers"). Override `isEtbTriggerGate()` when adding a new
intervening-if condition used to gate a targeted ETB.
- `ConditionalEffect(new ControllerTurn(), CardEffect)` ŌĆö during your turn
- `ConditionalEffect(new NotControllerTurn(), CardEffect)` ŌĆö during turns other than yours
- `ConditionalEffect(new ControlsPermanent(filter), wrapped)` ŌĆö controls matching
- `EnchantedPermanentConditionalEffect(PermanentPredicate, CardEffect ifMatch, CardEffect ifNotMatch)` ŌĆö aura active branch based on enchanted permanent predicate
- `ConditionalEffect(new OpponentControlsPermanent(filter), wrapped)` ŌĆö opponent controls matching
- `ConditionalEffect(new OpponentControlsMoreLands(), wrapped)` ŌĆö an opponent controls strictly more lands than you (Gift of Estates)
- `ConditionalEffect(new HasAttacker(predicate), wrapped)` ŌĆö one or more matching attackers
- `CantAttackUnlessEffect(Condition, "unless clause")` ŌĆö STATIC attack restriction; condition = `ControlsPermanentCount(1, filter)` / `DefendingPlayerControlsPermanent(filter)` / `AnyPlayerControlsPermanentCount(N, filter)` / `DefendingPlayerPoisoned()` / `OpponentDealtDamageThisTurn(minAmount)`
- `CantBlockUnlessEffect(Condition, "unless clause")` ŌĆö STATIC block-only restriction (Marauding Boneslasher: `new ControlsOtherPermanentCount(1, new PermanentHasSubtypePredicate(ZOMBIE))`, "you control another Zombie"). Block counterpart of `CantAttackUnlessEffect`; leaves attacking unrestricted, unlike `CantAttackOrBlockUnlessEffect`. Evaluated in `GameQueryService.canBlock`
  - `ControlsOtherPermanentCount(N, filter)` ŌĆö controller controls at least N matching permanents **excluding the source** (count analogue of `ControlsAnotherPermanent`; e.g. Amonkhet gods' `CantAttackOrBlockUnlessEffect(new ControlsOtherPermanentCount(3, new PermanentIsCreaturePredicate()), "you control at least three other creatures")`)
- `CantAttackOrBlockUnlessGreaterPowerAlsoDoesEffect()` ŌĆö STATIC combat-set restriction (Okk): can't attack unless another declared attacker has strictly greater power; can't block unless another declared blocker has strictly greater power. Validated against the current combat's declared set in `CombatAttackService`/`CombatBlockService`, not via `Condition`
- `CantAttackOrBlockUnlessCountAlsoDoesEffect(int otherCount)` ŌĆö STATIC combat-set restriction (Orcish Conscripts, `otherCount = 2`): can't attack unless at least N other creatures are declared as attackers; can't block unless at least N other creatures are declared as blockers. Validated against the current combat's declared set in `CombatAttackService`/`CombatBlockService`, not via `Condition`
- `CreaturesCantAttackUnlessPredicateEffect(PermanentPredicate exemption)` ŌĆö STATIC global: no creature can attack unless it matches exemption (Stormtide Leviathan)
- `CreaturesCantAttackControllerUnlessPredicateEffect(PermanentPredicate exemption[, boolean protectsPlaneswalkers])` ŌĆö STATIC defender-scoped: creatures not matching exemption can't attack THIS controller only ("creatures without flying can't attack you", Form of the Dragon ŌåÆ exemption `PermanentHasKeywordPredicate(Keyword.FLYING)`). Pass `protectsPlaneswalkers=true` to also bar attacks on the controller's planeswalkers ("can't attack you or planeswalkers you control", Sandwurm Convergence ŌåÆ exemption `PermanentNotPredicate(PermanentHasKeywordPredicate(Keyword.FLYING))`); the single-arg overload defaults to player-only
- `RequirePaymentToAttackEffect(int amountPerAttacker)` ŌĆö STATIC uniform attack tax read from the DEFENDER's battlefield; controller pays N per attacker (Windborn Muse / Ghostly Prison)
- `EnchantedCreatureCantAttackUnlessPaysEffect(int amount)` ŌĆö Aura STATIC: the enchanted creature can't attack unless its controller pays {amount} generic (Brainwash, {3}). Per-creature ŌĆö summed per declared attacker in `CombatAttackService`; contrast the uniform `RequirePaymentToAttackEffect`
- `CantAttackUnlessSacrificeEffect(count, PermanentPredicate, description)` ŌĆö STATIC additional attack cost: this creature can't attack unless its controller sacrifices `count` matching permanents (paid as attackers declared). Marker read by `AttackSacrificeCostService` (no dispatch handler). **Pair with** a `CantAttackUnlessEffect(new ControlsPermanentCount(count, filter), ŌĆ”)` legality gate so the declaration is only allowed when the cost is payable (Leviathan ŌĆö `(2, Island)`, "can't attack unless you sacrifice two Islands")
- `CantBlockHighPowerCreaturesUnlessPaysEffect(int minAttackerPower, int amount)` ŌĆö STATIC block-side tax (Hipparion, `(3, 1)`): this creature can't block a creature with power Ōēź `minAttackerPower` unless its controller pays {amount} generic. ## Damage

> **Power-based damage convention.** Any effect that deals damage equal to a creature's power
> (fight, bite, Pack Hunt, Berserker, Arc-Lightning-style source damage, planeswalker
> power-to-loyalty, `TargetDealsPowerDamageToTargetEffect`,
> `FightTargetsEffect`, `MassFightTargetCreatureEffect`,
> `SourceFightsTargetCreatureEffect`, the `SourcePower` dynamic amount,
> `PackHuntEffect`) must read the amount via
> `gameQueryService.getPowerBasedDamage(gameData, source)` ŌĆö **never** via
> `getEffectivePower` with a manual `> 0` guard. The helper clamps negative power to 0 per
> CR 510.1a so the damage primitives never see negative values.

- `DealDamageToAnyTargetEffect(DynamicAmount, boolean cantRegenerate, boolean exileInsteadOfDie)`; `(int)`, `(int, boolean)`, `(DynamicAmount)` ŌĆö any target. Amounts: `Fixed`, `XValue` (X spells / cost-snapshotted power), `SourcePower`, `CountersOnSource(CHARGE)`, ŌĆ”
- `DealDamageToSourceEffect(DynamicAmount)`; `(int)` ŌĆö the source permanent deals damage to itself (no target). Pair with `DealDamageToAnyTargetEffect` for "deals X to any target and X to itself" (Sunflare Shaman)
- `DealDamageToAttackedTargetEffect(int damage)` ŌĆö damage to the player or planeswalker attacked by the creature that caused the attack trigger
- `EquippedCreatureDealsPowerToAttackedTargetEffect()` ŌĆö Equipment `ON_ATTACK`: equipped creature deals damage equal to its power to the player/planeswalker it's attacking (Mage Slayer); non-targeting, equipped creature is the damage source

### Sacrifice costs

- `ExileSelfCost()` ŌĆö exile self as cost
- `SacrificeSelfCost()` / `(boolean trackPower)` ŌĆö sacrifice self as cost; `trackPower=true` snapshots effective power into xValue at payment (Mausoleum Wanderer + `CounterUnlessPaysEffect(0, true, false)`)

- `RemoveAllCountersFromTargetPermanentEffect(CounterType)` ŌĆö TARGET analog of the above: remove every counter of that type from the target permanent (benign `TargetCategory.CREATURE`; card's target filter narrows further), no-op if none. - `SacrificeArtifactCost()` ŌĆö sacrifice artifact
- `SacrificePermanentCost(PermanentPredicate, String[, excludeSource])` ŌĆö sacrifice matching permanent; use creature+subtype predicates with `excludeSource=false` for source-eligible "sacrifice a [subtype]"
- `SacrificePermanentsSequenceCost(List<PermanentPredicate> filters, List<String> descriptions)` ŌĆö sacrifice one distinct permanent per per-slot filter, in order (Angel's Herald: "a green creature, a white creature, and a blue creature"). Dead-end-safe (only offers picks that keep the remaining slots matchable)
- `DiscardCardTypeCost(CardPredicate, String)` ŌĆö discard matching card
- `DiscardRandomCardCost()` ŌĆö discard one card at random from hand (no choice); requires a non-empty hand (Coral Helm)
- `RemoveCounterFromSourceCost(int, CounterType)` ŌĆö remove counters from self
- `RemoveCounterFromSourceEffect(CounterType, int amount)` ŌĆö resolved (not a cost): remove up to `amount` counters of a type from the SOURCE permanent, clamped at zero (no-op if none). Self-targeting, so trigger collectors carry the source id. - `RemoveCounterFromTargetAndGainLifeEffect(CounterType, int lifeGain)` ŌĆö remove one counter of a type from target permanent; gain `lifeGain` life only if a counter was removed ("If you do") (Woeleecher: `MINUS_ONE_MINUS_ONE`, 2)
- `RemoveAndPutCounterOnAttackEffect(CounterType)` / `()` (=`MINUS_ONE_MINUS_ONE`) ŌĆö **ON_ATTACK only, two targets**: remove one counter of the type from the first target (a creature you control; no-op if none) and put one of the same type on the second (up to one creature the defending player controls; fires the put triggers + respects can't-have). Independent halves (puts even with nothing to remove). Implements the marker `AttackCounterMoveEffect`, so `CombatAttackService` routes it through the bespoke two-step `AttackCounterMoveFirstTarget`/`AttackCounterMoveSecondTarget` pipeline instead of the single-target attack pipeline (which can't collect two targets). Reads `entry.getTargetIds()[0]`/`[1]`. Decimator Beetle
- `RemoveCounterFromSourceAndGainLifeEffect(CounterType, int lifeGain)` ŌĆö SOURCE analogue of the above: remove one counter from this permanent; gain `lifeGain` life only if a counter was removed ("If you do"). Non-targeting; wrap in `MayEffect` for the "you may" form (Living Artifact: `VITALITY`, 1, on `UPKEEP_TRIGGERED`)
- `BountyOfTheLuxaEffect()` ŌĆö one atomic `PRECOMBAT_MAIN_TRIGGERED` toggle: remove all `FLOOD` counters from the source; if none were removed, put a flood counter on it and draw a card, otherwise add `{C}{G}{U}`. Because a flood counter is only ever added one at a time the card alternates draw/ramp each turn. Card-specific (hardcodes flood/draw-1/`{C}{G}{U}`); kept as one record because trigger collectors push one entry per slot effect so the counter mutation and its two branches can't be split. NOT a `ManaProducingEffect` ŌĆö a beginning-of-step trigger uses the stack (CR 605.1b), not a mana ability (Bounty of the Luxa)
- `RemoveCounterFromTargetPermanentEffect()` ŌĆö remove one counter of any kind currently on target permanent (first present type when several); no-op if none. "Remove a counter from target permanent" (Medicine Runner). Overload `(CounterType, PermanentPredicate)` removes one counter of exactly that type (no-op if the target has none of it) and narrows legal targets to permanents matching the predicate ŌĆö "remove a -1/-1 counter from another target creature you control" (Defiant Greatmaw: `MINUS_ONE_MINUS_ONE`, `AllOf(IsCreature, ControlledBySourceController, Not(IsSourceCard))`)
- `CrewCost(int)` ŌĆö crew
- `TapCreatureCost(PermanentPredicate)` ŌĆö tap creature
- `PayLifeCost(int)` ŌĆö pay life; `PayLifeCost.halfLife()` pays half your life rounded up

- `PutCounterOnControlledCreatureCost(CounterType, int count)` ŌĆö additional spell cost: put counter(s) on a creature you control (Scarscale Ritual: `MINUS_ONE_MINUS_ONE, 1`); creature supplied via `sacrificePermanentId`, paid in `SpellCastingService`

Unlike a plain `MayEffect(DrawCardEffect())`, which offers the draw to the source's controller
- `DyingCreatureControllerDiscardsCardEffect()` ŌĆö ON_ANY_CREATURE_DIES marker: whenever a qualifying creature dies, the DYING creature's controller (may be an opponent) discards a card (mandatory). Bereavement wraps it in `TriggeringCardConditionalEffect(CardColorPredicate(GREEN), ŌĆ”)` for "a green creature dies"
- Loot ("draw N, then discard M") ŌĆö _no dedicated effect_: compose `DrawCardEffect(N)` + `DiscardEffect(M, DiscardRecipient.CONTROLLER)` (flat pair on a spell/ability effect list ŌĆö one shared entry, so the draw resolves before the discard). - `DiscardAndDrawCardEffect(int discard, int draw)` ŌĆö rummage
- `DrawDiscardBoostSelfByDiscardedManaValueEffect()` ŌĆö draw 1, discard 1, then source gets +X/+0 until end of turn where X = the discarded card's mana value (Spellbound Dragon, `ON_ATTACK`). Amount: `EventValue()` for "equal to the life you gained" (Sanguine Bond `(new EventValue(), TARGET_PLAYER)`); `PermanentCount(filter, CONTROLLER)` for "1 life for each ŌĆ” you control" (Bishop); `new XValue()` for Exsanguinate `(new XValue(), EACH_OPPONENT, true)`. the spec targets a player iff `recipient == TARGET_PLAYER`
- "target player loses N life and you gain M life" (fixed drain) ŌĆö **compose** `LoseLifeEffect(N, LoseLifeRecipient.TARGET_PLAYER)` + `GainLifeEffect(M)` as two effects on one slot; targeting auto-derives from the loss, controller gains the fixed M on resolution. Omit the `GainLifeEffect` for a pure "target player loses N life". The `TARGET_PLAYER` loss does NOT fire "loses life" triggers (pre-existing quirk). Syphon Life, Soul Feast, Bleak Coven Vampires
- `GainLifeEqualToTargetCreaturePowerEffect()` ŌĆö controller gains life equal to a single target creature's effective power. Declares its own `TargetCategory.CREATURE` spec (unlike `GainLifeEffect(new TargetPower())`, whose target is supplied by a co-effect like Chastise's destroy), so it stands alone as a targeted trigger. `LifeGainEffect`. Wrap in `MayEffect` and give the card a `target(PermanentPredicateTargetFilter(creature + PermanentControlledBySourceControllerPredicate))` for "you may gain life equal to the power of target creature you control" (Wall of Reverence, `CONTROLLER_END_STEP_TRIGGERED`; the target is chosen at resolution via the queueMayAbility path since the End-step pipeline routes a `MayEffect` there)
- `SpellCastLifeDrainEffect(int lifeLoss, int lifeGain, CardPredicate spellFilter)` ŌĆö `ON_OPPONENT_CASTS_SPELL` drain: opponent who casts a matching spell loses `lifeLoss`, you gain `lifeGain` (filter null = any). Yawgmoth's Edict
- `LoseLifeUnlessPaysEffect(int lifeLoss, int payAmount, CardPredicate spellFilter)` ŌĆö `ON_OPPONENT_CASTS_SPELL` punisher: casting opponent loses `lifeLoss` life unless they pay `{payAmount}` (filter null = any). Can't-pay auto-loses; else may-ability prompt. Isolation Cell (2, 2, CREATURE)
- `DamageUnlessPaysEffect(int damage, int payAmount, CardPredicate spellFilter)` ŌĆö `ON_OPPONENT_CASTS_SPELL` punisher, **damage** version of `LoseLifeUnlessPaysEffect` (goes through the damage system: preventable, infect, redirect). `DamageDealingEffect`. Can't-pay auto-deals; else may-ability prompt. Soul Barrier (2, 2, CREATURE)
- `RegisterLoseLifeAtNextDrawStepUnlessPaysEffect(int lifeLoss, int payAmount)` ŌĆö `ON_DAMAGE_TO_PLAYER` trigger (Nafs Asp, `(1, 1)`): schedules a `LoseLifeAtNextDrawStepUnlessPays` delayed action on the damaged player (baked in as `targetId` via `DAMAGED_PLAYER`); at the beginning of that player's next draw step `StepTriggerService` drains it into a `ForcedCostOrElseEffect` pay-or-lose-life may-prompt (pay `{payAmount}` ŌåÆ no loss, else lose `lifeLoss`). Life loss, not damage ŌĆö won't re-trigger
- `TormentOfHailfireEffect(int lifeLoss)` ŌĆö SPELL, X-times repeated three-way punisher (Torment of Hailfire, `3`): "Repeat X times. Each opponent loses N life unless they sacrifice a nonland permanent or discards a card." X = xValue. Per opponent per iteration (APNAP), a pruned `ColorChoice` list-pick (lose life inline / sacrifice a nonland permanent / discard a card); life is always offered. One-opponent-at-a-time loop over `GameData.torment`, re-running via `rerunCurrentEffectAfterInteraction` (same driver shape as `EachPlayerMayDiscardUpToThenTakeDamageEffect`/`IllicitAuctionEffect`). No target
- `PlayersCantGainLifeEffect()` ŌĆö can't gain life (static)
- `CountersCantBePlacedEffect()` ŌĆö STATIC global (Solemnity): players can't get counters and counters can't be put on artifacts/creatures/enchantments/lands (planeswalkers keep loyalty). Queried at `GameQueryService.cantHaveCounters` + `canPlayerGetPoisonCounters`
- `TargetPlayerCantGainLifeRestOfGameEffect()` ŌĆö the stack entry's target player can't gain life for the rest of the game (persistent, per-player). Non-targeting on `ON_DAMAGE_TO_PLAYER`; Stigma Lasher
- `AllDamageDealtWithWitherEffect()` ŌĆö STATIC global: all damage is dealt as though its source had wither (creature damage becomes -1/-1 counters; player damage normal). Everlasting Torment
- `NoncombatDamageToOpponentCreaturesAsMinusCountersEffect()` ŌĆö STATIC: the controller's sources deal noncombat damage to opponents' creatures as -1/-1 counters instead (combat damage unaffected). Like wither, still counts as damage dealt. Checked in `GameQueryService.noncombatDamageToOpponentCreatureAsCounters` (noncombat path only). Soul-Scar Mage
- `DoubleLifeGainEffect()` ŌĆö STATIC: controller's life gain is doubled (Boon Reflection). Applied in `LifeSupport.applyGainLife`; multiple copies stack multiplicatively (2^count)

## Creature pump / boost

- `BoostTargetCreatureEffect(DynamicAmount power, DynamicAmount toughness)` or `(int, int)` ŌĆö target +X/+Y. Any "for each ŌĆ”", "+X/+X" (X paid), or "where X is ŌĆ”" target-pump = this effect + a `model/amount/DynamicAmount` ŌĆö never a new per-variant class. The amount evaluates against the SOURCE, so counting refers to the effect's controller, not the pumped target. E.g. `(new XValue(), new XValue())` (Untamed Might), `(new PermanentCount(new PermanentIsCreaturePredicate(), CountScope.CONTROLLER), same)` (Elder of Laurels), `(new Sum(new Fixed(1), new CardsInGraveyard(filter, CountScope.CONTROLLER)), new Fixed(0))` (Ancestral Anger)
- `BuffTargetCreatureIndefinitelyEffect(int power, int toughness, Set<Keyword> keywords)` or `(int, int)` ŌĆö target creature gets +power/+toughness and gains `keywords` **indefinitely** (no duration, CR 611.2b). Use for "this effect lasts indefinitely" pumps (Riding the Dilu Horse); NOT for until-EOT pumps (use `BoostTargetCreatureEffect` + `GrantKeywordEffect`). Recorded as a `PERMANENT` floating continuous effect on the target ŌĆö +P/+T in sublayer 7c, keywords in layer 6 (read off the float by `GameQueryService.assembleStaticBonus`); copies stack additively
- `BoostTargetCreatureWhileSourceTappedEffect(int power, int toughness)` ŌĆö target creature gets +power/+toughness **for as long as the source permanent remains tapped** (CR 611.2b; Tawnos's Weaponry). Records a `WHILE_SOURCE_TAPPED` floating `BuffTargetCreatureIndefinitelyEffect` keyed to the source, read in sublayer 7c; expired when the source untaps (`CreatureControlService.onSourceUntapped` ŌåÆ `GameData.expireTappedSourceFloatingEffects`) or leaves. Pair with `MayNotUntapDuringUntapStepEffect` static + a `{T}` cost so the controller can keep the source tapped
- `CardNamedPredicate(String cardName)` ŌĆö card filter for exact name match (use with graveyard-count boosts above)
- `BoostSelfEffect(DynamicAmount, DynamicAmount)` or `(int, int)` ŌĆö self +X/+Y; one-shot in trigger/ability slots, continuous in STATIC. Any "for each ŌĆ”" self-boost = this effect + a `model/amount/DynamicAmount` (`PermanentCount`, `CardsInGraveyard`, `AttachmentsOnSource`, `CreaturesBlockingSource`, `OpponentPoisonCounters`, `ImprintedCreaturePower/Toughness`, `LandsMatchingImprintedName`, `ChosenPermanentPower`, `XValue`, `Scaled`, `Fixed`) ŌĆö never a new per-variant effect class. `ChosenPermanentPower` = effective power (at resolution) of the permanent chosen during activation, e.g. the creature tapped by `TapCreatureCost(ŌĆ”, trackTappedCreaturePower=true)` ŌĆö Impelled Giant's "+X/+0 where X is the power of the creature tapped this way"
- `AttachedBoostEffect(DynamicAmount, DynamicAmount, GrantScope)` ŌĆö STATIC +X/+Y on the enchanted/equipped creature (`ENCHANTED_CREATURE`/`EQUIPPED_CREATURE`). Attached-scope sibling of `BoostSelfEffect`; any "for each ŌĆ”" aura/equipment boost = this effect + a `DynamicAmount`. `CountScope.CONTROLLER` = the aura/equipment's controller (CR 109.5). Negative per-count = wrap in `Scaled(ŌĆ”, -1)`. Blanchwood Armor, Blackblade Reforged, Bonehoard, Runechanter's Pike, Quag Sickness, Strata Scythe ŌĆö never a new `BoostCreaturePer*` class
- `BoostSelfWhenCombatOpponentMatchesEffect(PermanentPredicate opponentFilter, int power, int toughness)` ŌĆö "Whenever this creature blocks or becomes blocked by one or more [filter] creatures, it gets +X/+Y until end of turn." Place in ON_BLOCK and/or ON_BECOMES_BLOCKED (plain, non-PER_BLOCKER). Non-targeting: the trigger always goes on the stack; the handler applies the boost only if a combat opponent (a creature this blocks, or a creature blocking it) matches the filter, and fires once regardless of how many match. - `DoubleSelfPowerToughnessEffect()` ŌĆö double self P/T
- `BoostAllOwnCreaturesEffect(DynamicAmount, DynamicAmount)` or `(ŌĆ”, PermanentPredicate)` ŌĆö all own +X/+Y; `(int, int[, PermanentPredicate])` convenience wraps in `Fixed`. Any "where X is ŌĆ”" / power- or graveyard-derived mass own-pump = this effect + a `DynamicAmount` (evaluated once at resolution) ŌĆö e.g. `new GreatestPowerAmongControlled()` (Overwhelming Stampede), `new CardsInGraveyard(new CardTypePredicate(CREATURE), CONTROLLER)` (Garruk, the Veil-Cursed). Never a new per-variant class
- `BoostAllCreaturesEffect(DynamicAmount, DynamicAmount)` or `(ŌĆ”, PermanentPredicate)` or `(ŌĆ”, PermanentPredicate, EachPermanentScope)` ŌĆö creatures +X/+Y; `(int, int[, PermanentPredicate])` / `(int, int, EachPermanentScope)` convenience wraps in `Fixed`. Scope `ALL_PLAYERS` (default, both sides) or `TARGET_PLAYER` ("creatures target player controls", player-targeting spec, Shields of Velis Vel). - `StaticBoostEffect(int, int, Set<Keyword>, GrantScope, PermanentPredicate)` ŌĆö static +X/+Y + keywords
- `BoostOwnCreaturesByManaSymbolEffect(ManaColor, int powerPerSymbol, int toughnessPerSymbol)` ŌĆö chroma anthem: each creature you control gets +P/+T per mana symbol of that color in its own cost (hybrid/Phyrexian symbols of that color count). Light from Within
- `BoostOtherMulticoloredCreaturesByColorCountEffect(int powerPerColor, int toughnessPerColor)` ŌĆö anthem: each *other* multicolored creature you control gets +P/+T per color it has. - `SetBasePowerToughnessEffect(int, int)` ŌĆö set target creature's base P/T until end of turn; `(int, int, GrantScope)` for continuous static (e.g. `ENCHANTED_CREATURE`, Deep Freeze)
- `SetAllOwnCreaturesBasePowerToughnessEffect(DynamicAmount, DynamicAmount)` or `(int, int)` ŌĆö set base P/T of all creatures you control to X/X until end of turn (layer 7b, modifiers apply on top). X-cost ability = `new XValue()` (Mirror Entity)
- `SetAllUnblockedCreaturesBasePowerToughnessEffect(int, int)` ŌĆö set base P/T of every unblocked creature (any player's) until end of turn (layer 7b). "Unblocked" = attacking + no blocker, locked in at resolution (Inkfathom Witch)
- `GrantBaseStatsToCounterBearersEffect(CounterType, int power, int toughness, Set<Keyword>)` ŌĆö resolves to a game-wide continuous rule: "every creature with a [type] counter has base P/T power/toughness and has [keywords] for as long as it has such a counter" (layer 7b set + layer 6 keywords). ## Keywords / abilities

- `GrantKeywordEffect(Keyword, GrantScope)` or `(Keyword, GrantScope, PermanentPredicate)` or `(Set<Keyword>, GrantScope)` ŌĆö grant keywords. Add a trailing `GrantDuration` (`(Keyword, GrantScope, GrantDuration)` / `(Set<Keyword>, GrantScope, GrantDuration)`) for one-shot duration: `END_OF_TURN` (default) or `UNTIL_YOUR_NEXT_TURN`. In `STATIC` slot the grant is continuous and the duration is ignored. `GrantKeywordEffect.toTargetIf(Keyword, PermanentPredicate grantCondition)` ŌĆö grant to target ONLY if it matches `grantCondition` (checked at resolution; target stays legal, only the grant is conditional ŌĆö Vampire's Zeal, Blessing of Belzenlok). `grantCondition` is distinct from `filter`/the spec predicate.
- `GrantEffectToOwnCreaturesUntilEndOfTurnEffect(EffectSlot, CardEffect)` ŌĆö snapshot grant of a temporary triggered ability to creatures you control until EOT (later ETBs/control changes excluded). Driven/Despair (`ON_COMBAT_DAMAGE_TO_PLAYER` + draw / discard). Own-creatures analog of `GrantEffectToTargetUntilEndOfTurnEffect`
- `RemoveKeywordEffect(Keyword, GrantScope)` or `(Keyword, GrantScope, PermanentPredicate filter)` ŌĆö one-shot keyword removal until end of turn (floating layer-6 removal). Pass an `EffectDuration` ŌĆö `(Keyword, GrantScope, EffectDuration)` or `(Keyword, GrantScope, filter, EffectDuration)` ŌĆö for a non-end-of-turn loss; `EffectDuration.PERMANENT` = indefinite loss that lasts as long as the permanent stays on the battlefield (Elder Land Wurm's "When this creature blocks, it loses defender", ON_BLOCK). - `PreventTargetCreatureRegenerationThisTurnEffect()` ŌĆö "target creature can't be regenerated this turn": sets `cantRegenerateThisTurn` on the target so `GraveyardService.tryRegenerate` refuses it for the rest of the turn (beats shields AND intrinsic regen); cleared at turn cleanup. `targetSpec()` narrows to a CREATURE (harmful). - `ProtectionFromColorsEffect(Set<CardColor>)` ŌĆö protection from colors (static)
- `ProtectionFromSubtypesEffect(Set<CardSubtype>)` ŌĆö protection from subtypes (static)
- `ProtectionFromManaValueEffect(int minManaValue)` ŌĆö protection from sources with mana value Ōēź N (static, Mistmeadow Skulk)
- `ProtectionFromEverythingEffect()` ŌĆö protection from everything: every source, any colour/type/subtype/mana value (static, Progenitus)

- `TargetCreatureBecomesSubtypeUntilEndOfTurnEffect(CardSubtype)` ŌĆö target creature **becomes** the given creature type until end of turn, **replacing** all its other creature types (Boldwyr Intimidator: "target creature becomes a Coward"). - `SourceBecomesSubtypeUntilEndOfTurnEffect(CardSubtype)` ŌĆö self analog (no target): the **source** permanent becomes the given creature type until end of turn, replacing all its other creature types (Paragon of the Amesha: "this creature becomes an Angel"). - `GrantBasicLandTypeToTargetEffect(EffectDuration[, CardSubtype fixedSubtype][, boolean replacing])` ŌĆö target land becomes a chosen basic land type. Default adds "in addition to its other types" (Navigator's Compass / Aquitect's Will); `replacing=true` makes the land **become** the type, losing its others per rule 305.7 (Tideshaper Mystic, UNTIL_END_OF_TURN; Orcish Farmer = fixed SWAMP, UNTIL_CONTROLLERS_NEXT_UNTAP_STEP ŌĆö survives end of turn, cleared at the land controller's next untap step)
- `NonbasicLandsBecomeTypeEffect(CardSubtype)` ŌĆö STATIC, global: every nonbasic land (any controller) becomes the basic land type, losing its other land types/abilities and producing that type's mana per rule 305.7. Basic lands unaffected. Blood Moon (MOUNTAIN)
- `LandsOfSubtypeBecomeTypeEffect(CardSubtype from, CardSubtype to)` ŌĆö STATIC, global: every land (any controller, basic or nonbasic) carrying the `from` land subtype becomes the `to` basic land type, losing its other land types/abilities and producing that type's mana per rule 305.7. - `OwnLandsBecomeChosenTypeUntilEndOfTurnEffect()` ŌĆö no target: prompts the controller for a basic land type, then each land they control **becomes** that type until end of turn (type-replacing per rule 305.7, reuses `applyBasicLandType(..., replacing=true)`). Applied once at resolution to lands controlled then. Elsewhere Flask
- `TargetLandBecomesForestUntilSourceLeavesEffect()` (ability, benign LAND) + `TrackedLandsBecomeForestEffect()` (STATIC) ŌĆö Gaea's Liege's "{T}: Target land becomes a Forest until this creature leaves the battlefield". - `LoseAllCreatureTypesEffect(GrantScope)` ŌĆö creatures lose all creature types until end of turn; `TARGET` = single creature (Amoeboid Changeling), `TARGET_PLAYERS_CREATURES` = all creatures target player controls (Ego Erasure); "gains all creature types" = `GrantKeywordEffect(Keyword.CHANGELING, sameScope)`
- **Paradigm** (`Keyword.PARADIGM` on card, not an effect) ŌĆö engine handled by `ParadigmService`: first resolve exiles spell + registers `GameData.ParadigmDelayedTrigger`; each precombat main fires `ParadigmCastCopyEffect` ŌåÆ copy in exile + `ParadigmMayCastFromExileEffect` may-cast (`ParadigmCastSupport`)

- `CantBeBlockedEffect()` ŌĆö unblockable (static)

One-shot analogue of static `MatchingCreaturesMustAttackEffect`. - `TargetPlayersCreaturesMustAttackThisTurnEffect()` ŌĆö targets a player: every creature that player controls must attack this turn if able (may attack any legal target ŌĆö not "attack you"). Sets transient `mustAttackThisTurn` on each of the target player's creatures at resolution, cleared at end of turn. Player-targeted analogue of the non-targeted `MatchingCreaturesMustAttackThisTurnEffect`; SPELL slot, `benign(PLAYER)` spec. Imaginary Threats
- `MustAttackControllerNextTurnEffect()` ŌĆö targets a player: during their next turn every creature they control attacks you (the controller) if able (Taunt). SPELL slot; player-targeting spec
- `SirensCallEffect()` ŌĆö SPELL, non-targeted: every creature the active player controls must attack this turn if able, and at the next end step all non-Wall creatures they control that didn't attack (and weren't summoning sick) are destroyed. Siren's Call. Pair with `setSpellCastTimingRestriction(OPPONENTS_TURN_BEFORE_ATTACKERS)`. Queues a `DestroyNonAttackersAtEndStep` delayed action
- `MustBeBlockedIfAbleEffect()` ŌĆö must be blocked (static)
- `MustBeBlockedByAllCreaturesEffect()` ŌĆö Lure (static)
- `MustBeBlockedByAllCreaturesThisTurnEffect()` ŌĆö one-shot targeted Lure: all creatures able to block target creature this turn do so (Alluring Scent); SPELL slot, permanent-targeting spec
- `MustBlockTargetCreatureEffect()` ŌĆö two-target spell: blocker group (0) must block blocked group (1) this turn if able (Hunt Down)
- `MustBlockThisTurnIfAbleEffect()` ŌĆö single-target: target creature must be declared as a blocker if it can block any attacker (general "blocks this turn if able", no specific attacker ŌĆö unlike Provoke's `MustBlockSourceEffect`). - `MakeCreatureUnblockableEffect()` ŌĆö target unblockable this turn
- `CanBeBlockedOnlyByFilterEffect(PermanentPredicate blockerPredicate, String allowedBlockersDescription)` ŌĆö static evasion on the source: it can be blocked only by blockers matching the predicate (Fear-like, e.g. Dread Warlock = black creatures)

Dread Charge = both filters `PermanentColorInPredicate(BLACK)`. Affected creatures snapshotted at resolution; restriction stored transiently on each `Permanent`
- `MakeCreatureBlockableOnlyByFilterThisTurnEffect(PermanentPredicate blockerPredicate, String allowedBlockersDescription)` / `(ŌĆ”, true)` ŌĆö one-shot: creature can be blocked only by blockers matching `blockerPredicate` until end of turn. Default targets a creature (Joven's Tools = `PermanentHasSubtypePredicate(WALL)`, "Walls"); pass `selfTargeting=true` for "this creature" trigger payoffs (Rhonas's Stalwart: `PermanentPowerAtLeastPredicate(3)` for "can't be blocked by power 2 or less"). Sibling of `GrantCanBeBlockedOnlyByFilterToOwnCreaturesEffect`
- `MatchingCreaturesCantBlockMatchingCreaturesEffect(PermanentPredicate blockerPredicate, PermanentPredicate attackerPredicate, String description)` ŌĆö global STATIC: while the source is on the battlefield, any creature matching `blockerPredicate` can't block any creature matching `attackerPredicate`, board-wide (Boldwyr Intimidator: "Cowards can't block Warriors"). - `MatchingCreaturesCantAttackOrBlockEffect(PermanentPredicate affectedPredicate, String description)` ŌĆö global STATIC: while the source is on the battlefield, any creature matching `affectedPredicate` can't attack OR block, board-wide. NOT the static `CantBlockEffect()`.
- `TargetPlayerChoosesCreatureRestCantBlockEffect()` ŌĆö SPELL, player-targeting spec: the targeted player chooses one creature they control (kept able to block); all their OTHER creatures can't block this turn. ## Tap / untap

- `EnchantedCreatureDealsDamageEqualToDealtDamageToControllerEffect()` ŌĆö deals xValue damage to the enchanted creature's controller (ON_ENCHANTED_CREATURE_DEALT_DAMAGE = damage it was dealt, Spiteful Shadows; ON_ENCHANTED_CREATURE_DEALS_DAMAGE_TO_YOU = damage it dealt to the Aura's controller, Backfire)
- `ReflectSourceDamageToItsControllerEffect(CardColor.RED)` ŌĆö global watcher (ON_ANY_SOURCE_DEALS_DAMAGE): whenever any red creature or spell deals damage, this permanent deals that much to that source's controller (Justice). Damage from one source is summed into one reflection (CR ruling)
- `TapPermanentsEffect(TapUntapScope.TARGET)` ŌĆö tap target
- `TapTargetCreatureUnlessControllerPaysLifeEffect(int lifeCost)` ŌĆö tap target creature unless its controller pays N life (Vectis Dominator, 2). Punisher: the **target creature's controller** decides (pay-life vs. tap; can't-pay auto-taps). Ability-level creature `target(...)` supplies the filter
- `TapPermanentsEffect(TapUntapScope.SELF)` ŌĆö tap self ┬Ę `.ENCHANTED` ŌĆö tap aura's enchanted creature
- `TapPermanentsEffect(TapUntapScope.TARGET_PLAYERS_PERMANENTS, filter)` ŌĆö tap that player's matching permanents
- `TapPermanentsEffect(TapUntapScope.ALL_CREATURES, filter)` ŌĆö tap all creatures matching filter (`PermanentIsAttackingPredicate` = all attackers)
- `UntapPermanentsEffect(TapUntapScope.TARGET[, PermanentPredicate])` ŌĆö untap target (predicate restricts targets)
- `UntapPermanentsEffect(TapUntapScope.SELF)` ŌĆö untap self ┬Ę `.ALL_TARGETS` ŌĆö untap all targets
- `UntapPermanentsEffect(TapUntapScope.CONTROLLED, filter)` ŌĆö untap all you control matching ┬Ę `.OTHER_CONTROLLED_CREATURES` ŌĆö untap each other creature you control ┬Ę `.ATTACKED_CREATURES` ŌĆö untap creatures that attacked this turn ┬Ę `.ALL_CREATURES[, filter]` ŌĆö untap every creature on every battlefield matching filter (null = all creatures); Intruder Alarm
- `UntapPermanentsEffect(TapUntapScope.TARGET_PLAYERS_PERMANENTS, filter)` ŌĆö untap all of target player's permanents matching filter (Early Harvest: `PermanentAllOf(land, BASIC supertype)` = target player's basic lands)
- `UntapChosenPermanentEffect(PermanentPredicate)` ŌĆö "untap target [permanent]" as a **resolution-time controller choice** (not a cast-time target) for a triggered ability on a slot with no targeting pipeline, e.g. combat-damage triggers. Pushed non-targeting; at resolution the controller picks one matching permanent across every battlefield to untap via `MultiPermanentChoiceContext.UntapChosenPermanent`. Initiate's Companion (`ON_COMBAT_DAMAGE_TO_PLAYER`, `PermanentAnyOfPredicate(creature, land)`)
- `UntapEquippedCreatureEffect()` ŌĆö untap the source Equipment's attached creature (fizzles if unattached). Place on the Equipment in a trigger slot (e.g. `ON_ANY_CREATURE_DIES`) to model equipment-granted untap triggers; Thornbite Staff
- `MatchingPermanentsDoesntUntapEffect(PermanentPredicate)` ŌĆö global static: every permanent matching the predicate (any controller, incl. the source) doesn't untap during its controller's untap step; Marble Titan (`PermanentPowerAtLeastPredicate(3)`)
- `PayManaPerTappedCreatureToUntapEffect(manaPerCreature, PermanentPredicate)` ŌĆö `EACH_UPKEEP_TRIGGERED` (Magnetic Mountain): "that player may choose any number of tapped [matching] creatures they control and pay {manaPerCreature} for each; if they do, untap those." Acting player = stack entry's target (active player); at resolution they get a `MultiPermanentChoice` capped by affordability, then pay `manaPerCreature`├Śchosen from their pool and untap (`MultiPermanentChoiceContext.PayManaPerCreatureUntap`, handled in `MultiPermanentChoiceHandlerService`). Pair with STATIC `MatchingPermanentsDoesntUntapEffect(sameFilter)`
- `StorageMatrixEffect()` ŌĆö global static (Storage Matrix): while any permanent carrying it is untapped, each player's untap step pauses (`UntapStepService.storageMatrixRestrictionApplies`) so the active player chooses artifact/creature/land (a `ColorChoice` with `ChoiceContext.StorageMatrixUntapChoice`); only permanents of the chosen type untap that step. No card-side targeting/config ŌĆö just `addEffect(EffectSlot.STATIC, new StorageMatrixEffect())`.
- `StaticOrbEffect(maxUntap, filter, requiresUntappedSource)` ŌĆö global "can't untap more than N [permanents]" lock (Static Orb, Stoic Angel). While an active copy is present (`requiresUntappedSource` ŌåÆ only while its source is untapped), each player's untap step pauses (`UntapStepService.bindingUntapRestriction`, only when more than `maxUntap` of the matching permanents would untap) so the active player picks up to `maxUntap` of the permanents that match `filter` (`null` = all) and would otherwise untap (a `MultiPermanentChoice` with `MultiPermanentChoiceContext.StaticOrbUntap`); only those, plus any permanents the filter excludes, untap that step. Static Orb: `new StaticOrbEffect()` (cap 2, all permanents, while untapped). Stoic Angel and Smoke: `new StaticOrbEffect(1, new PermanentIsCreaturePredicate(), false)` (cap 1, creatures only, always-on). Winter Orb: `new StaticOrbEffect(1, new PermanentIsLandPredicate(), true)` (cap 1, lands only, while untapped).
- `DoesntUntapEffect.self()` ŌĆö this permanent doesn't untap (static) ┬Ę `.enchanted()` ŌĆö attached aura/equipment's host doesn't untap (static) ┬Ę `.targetWhileSourceOnBattlefield()` ŌĆö target doesn't untap while source on battlefield (Dungeon Geists / Time of Ice) ┬Ę `.targetWhileSourceTapped()` ŌĆö while source stays tapped (Rust Tick); TARGET factories piggyback on a companion `TapPermanentsEffect(TapUntapScope.TARGET)`
- `SkipNextUntapEffect(TapUntapScope.TARGET)` ŌĆö target permanent skips next untap (piggybacks on companion targeting effect) ┬Ę `.SELF` ŌĆö source permanent itself skips next untap, non-targeting, for self-referential triggers like `ON_ATTACK` (Lead Golem) ┬Ę `.CONTROLLED, filter` ŌĆö the controller's own matching permanents, non-targeting ("Lands you control don't untap during your next untap step" = `PermanentIsLandPredicate`, Oketra's Last Mercy) ┬Ę `.TARGET_PLAYERS_PERMANENTS, filter` ŌĆö that player's matching permanents ┬Ę `.ALL_CREATURES, filter` ŌĆö all creatures matching filter (`PermanentIsAttackingPredicate` = all attackers)
- `IfWonClashEffect(wrapped)` ŌĆö clash-only marker on `EffectSlot.ON_CONTROLLER_CLASHES`: the wrapped effect applies only if the controller won the clash ("If you won, ..."). Consumed by `TriggerCollectionService.fireClashTriggers` at trigger time (not a stack effect). See Entangling Trap: tap target opponent creature + `IfWonClashEffect(SkipNextUntapEffect(TARGET))`. Clash is performed via `performClash` (2-player: both reveal top card, strictly-higher mana value wins).
- `IfLostClashEffect(wrapped)` ŌĆö mirror of `IfWonClashEffect`: applies only when the controller did **not** win. Pair a won- and lost-variant so exactly one branch fires when the base effect happens regardless of outcome but only a detail differs (Rebellion of the Flamekin: two `MayPayManaEffect("{1}", CreateTokenEffect(...))` differing only in granted `HASTE` on the won branch). Non-targeting clash triggers go straight onto the stack.
- `ClashEffect(List<CardEffect> beforeClash, CardEffect onWin, boolean repeatWhileWinning)` ŌĆö the clash-*source* stack effect; convenience ctor `ClashEffect(onWin)` = `(List.of(), onWin, false)`. Each iteration dispatches `beforeClash` (via each effect's own handler, against the same entry), performs the clash for the controller via `performClash`, dispatches `onWin` on a win, and with `repeatWhileWinning` repeats the whole sequence until a lost clash (deck-out counts as a loss). "Clash with an opponent. If you win, [X]" = `ClashEffect(X)`; `onWin` may be null for a bare "clash with an opponent". Mirrors `FlipCoinWinEffect`. A multi-step win reward is a `SequenceEffect` ŌĆö expanded **synchronously** (each step via its own handler, against the same entry), so the steps must be synchronous (no async player-input pause between them). E.g. Oaken Brawler = `ClashEffect(new PutCountersOnSourceEffect(1, 1, 1))`. "[body], then clash with an opponent. If you win, repeat this process" = `ClashEffect(body, null, true)`: Hoarder's Greed = `ClashEffect(List.of(new LoseLifeEffect(2), new DrawCardEffect(2)), null, true)`. Wrap in `MayEffect` for "you may clash" (Sentry Oak = `MayEffect(ClashEffect(SequenceEffect.of(new BoostSelfEffect(2, 0), new RemoveKeywordEffect(Keyword.DEFENDER, GrantScope.SELF))), ...)`). Delegates its `targetSpec()` to `onWin`/`beforeClash`, so a **targeted** win reward works on any targeting slot: e.g. Springjack Knight "whenever this attacks, clash; if you win, target creature gains double strike" = `target(...)` + `ClashEffect(new GrantKeywordEffect(Keyword.DOUBLE_STRIKE, GrantScope.TARGET))` on `ON_ATTACK` (target chosen when the trigger goes on the stack; grant only on a win). Do **not** wrap an interactive `MayEffect` as a `ClashEffect` win reward ŌĆö the may-pause re-runs the `ClashEffect` (re-clash). For an *optional* win reward, use a bare `ClashEffect(null)` (records its result on the entry) followed by `ConditionalEffect(new WonClash(), new MayEffect(reward, prompt))`: Whirlpool Whelm = `ClashEffect(null)` + `ConditionalEffect(new WonClash(), new MayEffect(new PutTargetOnTopOfLibraryEffect(), prompt))` + `ReturnToHandEffect.target()`.

- `CantCastSpellTypeEffect(Set<CardType>)` / `(Set<CardType>, boolean appliesToAllPlayers)` ŌĆö can't cast spells of these types (static). Default controller-only (Steel Golem); pass `true` for symmetric "these spells can't be cast" (Aether Storm creature clause)
- `NoncreatureSpellsCantBeCastEffect(int minManaValue, boolean restrictXSpells)` ŌĆö global/symmetric: no player can cast a noncreature spell with mana value >= `minManaValue`, or (if `restrictXSpells`) with `{X}` in its cost (static, Gaddock Teeg `(4, true)`)
- `OpponentsCantCastSpellsWithManaValueAtMostEffect(int maxManaValue)` ŌĆö opponents of the source's controller can't cast spells with mana value Ōēż `maxManaValue` (static, Brisela `(3)`; chosen X counted at cast time)
- `CantCastAdditionalNonartifactSpellsEffect()` ŌĆö global/symmetric: each player who has already cast a nonartifact spell this turn can't cast additional nonartifact spells; artifact spells and each player's first nonartifact spell are unrestricted (static, Ethersworn Canonist; enforced in `CastingPermissionService.isAdditionalNonartifactSpellRestricted`)
- `AlternativeCostForSpellsEffect(String, CardPredicate)` (or `(String, CardPredicate, CounterType manaValueCapCounter, boolean oncePerTurn)`) ŌĆö alternative cast cost; optional counter-based mana-value cap + once-per-turn (As Foretold)
- `PlayersCantCastSpellsFromZonesEffect(Set<Zone> zones)` ŌĆö no player can cast from any zone in `zones` (static, global; only `GRAVEYARD`/`LIBRARY` enforced ŌĆö Ashes of the Abhorrent passes `Set.of(GRAVEYARD)`, Grafdigger's Cage passes `Set.of(GRAVEYARD, LIBRARY)`)
- `TargetPlayerCantPlayLandsThisTurnEffect()` ŌĆö target player can't play lands for the rest of this turn (spell; declare a player target via `target(...)`). Moonhold ({R} clause)
- `TargetPlayerCantCastCreatureSpellsThisTurnEffect()` ŌĆö target player can't cast creature spells for the rest of this turn (spell; declare a player target via `target(...)`). Moonhold ({W} clause)
- `WardOfBonesEffect()` ŌĆö static; each opponent controlling more creatures/artifacts/enchantments than the controller can't cast spells of that type (compared independently), and each opponent controlling more lands can't play lands. Controller never restricted. Ward of Bones
- `CardsCantEnterBattlefieldFromZonesEffect(CardPredicate filter, Set<Zone> zones)` ŌĆö cards matching `filter` (null = all) can't enter the battlefield from any zone in `zones`; blocks reanimation/undying/library-search-to-battlefield (static, global; only `GRAVEYARD`/`LIBRARY` enforced ŌĆö Grafdigger's Cage passes `CardTypePredicate(CREATURE)` and `Set.of(GRAVEYARD, LIBRARY)`)

## Choose / name

- `ChooseCardNameOnEnterEffect()` ŌĆö choose card name ETB
- `SpellsWithChosenNameCantBeCastEffect()` / `(boolean opponentsOnly)` ŌĆö STATIC; spells with the chosen name can't be cast. Default symmetric (Nevermore); `true` = opponents only, controller may still cast (Gideon's Intervention). Pair with `ChooseCardNameOnEnterEffect()`
- `PlayerHasProtectionFromChosenNameEffect()` ŌĆö STATIC; controller has protection from the chosen card name (Runed Halo); pair with `ChooseCardNameOnEnterEffect()`
- `PreventDamageFromChosenNameEffect()` ŌĆö STATIC; prevent all damage (combat + noncombat) to the controller AND permanents they control from sources with the chosen name ŌĆö damage only, not full protection (Gideon's Intervention). Pair with `ChooseCardNameOnEnterEffect()`; queried by `GameQueryService.isDamageFromChosenNamePreventedForController`
- `BoobyTrapEffect()` ŌĆö STATIC marker; chosen player reveals draws + name-match sac/10-damage trigger (Booby Trap), detected in DrawService
- `RevealFirstDrawDrawOnBasicLandEffect()` ŌĆö STATIC marker; controller reveals the first card they draw each turn, and if it's a basic land a "draw a card" trigger goes on the stack (Rowen), detected in DrawService (only the turn's first draw is revealed)
- `ChooseColorOnEnterEffect()` ŌĆö choose color ETB
- `AllNonlandPermanentsAreChosenColorEffect()` ŌĆö STATIC layer-5 color setter: all nonland permanents (any controller, incl. source) become the source's chosen color, replacing other colors. Pair with `ChooseColorOnEnterEffect` (Shifting Sky)
- `AllPermanentsGainChosenColorEffect()` ŌĆö STATIC layer-5 additive color grant: all permanents (any controller, including lands and the source) gain the source's chosen color *in addition to* their other colors (not replacing). Pair with `ChooseColorOnEnterEffect` (Painter's Servant). Battlefield permanents only ŌĆö does not recolor spells on the stack or cards in other zones
- `BecomeAllColorsUntilEndOfTurnEffect()` ŌĆö self-scoped layer-5 color set: the source permanent becomes all five colors until end of turn (no target/choice). Floats a `BecomeChosenColorsUntilEndOfTurnEffect` with every color on the source (Scrapbasket `{1}:`)
- `SetTargetColorEffect(CardColor)` ŌĆö SPELL; "target spell or permanent becomes [color]" **indefinitely** (Purelace ŌåÆ white). Layer-5 setter with `EffectDuration.PERMANENT` (no wear-off). Targets spell OR permanent like Glamerdye (spec `benign(PERMANENT)` + `targetsSpellOnStack`, no `target()` call); a color set on a permanent spell carries to the permanent via `GameData.spellColorOverrides` (CR 613.7)
- `ChooseSubtypeOnEnterEffect()` ŌĆö choose creature type ETB

## Provider map

All normal (stack-resolution) effects: one `NormalEffectHandlerBean` `@Component` per effect in `service/effect/normalfx/`, auto-registered by `GameEngineConfig`. Shared logic in `*Support` classes in the same package.

"You may ŌĆ”" accept/decline abilities follow the mirror pattern: one `MayEffectHandlerBean` `@Component` per effect in `service/effect/mayfx/` (declares `handledEffect()`, implements `handle(GameData, Player, boolean accepted, PendingMayAbility)`), auto-registered into `MayEffectHandlerRegistry` by `GameEngineConfig`. `MayAbilityHandlerService.handleMayAbilityChosen` runs its non-effect preambles, then iterates the ability's `effects()` in list order and dispatches to the first registered handler. The effect-keyed routing chain is now fully migrated; the residue in `MayAbilityHandlerService` is the generic accept/stack-construction path, the graveyard target-selection helpers, the CR 603.5 resolution-time may path, and Galvanoth's guarded `CastTopOfLibraryWithoutPayingManaCostEffect` branch ŌĆö all kept because they are entangled with target-selection/construction state, not because they are unmigrated routing. Beans that just orchestrate a sub-service (`MayCastHandlerService`, `MayMiscHandlerService`, `MayCopyHandlerService`, `MayPenaltyChoiceHandlerService`, `TariffSupport`) inject it; inline beans inject `GameBroadcastService`/`GameQueryService`/`InputCompletionService`/etc. directly. An interface-typed may-effect gets one bean per concrete implementor (registry keys on `effect.getClass()`), e.g. `CounterUnlessEffect` ŌåÆ `CounterUnlessPaysEffect` + `CounterUnlessDiscardsEffect`.
